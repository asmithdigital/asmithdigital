Degree Finder UX
University of Adelaide

During my employment at the University of Adelaide, I ran a UX research and design project for the design of the Degree Finder web application. I conducted user research using contextual enquiry, I analysed the data and created user stories, personas, red routes, usability requirements and information architecture. I then began iteratively designing wireframes and paper prototypes, working towards an improved user centered design solution.

The UX process

Build the context of use:

  • User research - contextual inquiry
  • Affinity diagramming
  • User mapping
  • Persona groups
  • Persona creation
  • Red routes
  • User stories

Create the user experience:

  • Information Architecture
  • Card sorting
  • Setting KPIs
  • Evaluate interaction design, rules, patterns etc
  • Paper prototype
  • Electronic prototype
  • Create usability test scenarios
  • Run usability tests
  • Usability data affinity diagramming
